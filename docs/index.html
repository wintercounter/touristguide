Welcome on your first site tour!

For first example: "This is the main menu where you can reach the main sections of the website."

Through this tour you'll find out how amazing this plugin is...

Simply with CSS Selectors

You can easily set the highlited element by any kind of CSS selectors. This "block" was selected with .figure img

Did you notice the animation? Oo
The plugin comes with an advanced, built-in animation engine. It has it's own syntax and you can create keyframes to have multiple moves in a single animation.

8 Positions

There are eight positions which you can use for you tooltips (top, right, bottom, left, topLeft, topRight, bottomLeft, bottomRight). Tooltip contents are coming from single divs, so you can use any kind of HTML in it, even embed videos if you need to.

Multiple Options

The plugin comes with many preferences and options to enable you high customization. In other way, the usage is very easy, you only need to have a few divs (the tooltip contents) and the JavaScript included.

You can customize your tooltips one-by-one with data attributes.

For example this tooltip uses the following attributes:
data-tg-for="#connect form"
data-tg-position="topRight"
data-tg-tip-offset="-5 30"
data-tg-overlay-offset="10"
data-tg-width="500"
data-tg-animation="[top: -5px;]....TOO LONG....|200[top: 5px;]

Help your visitors

Teach them how to use your site. Show them a video ;)

Smooth-Scroll

The plugin comes integrated with the ScrollTo plugin by Benjamin "balupton" Lupton. It helps to smoothly scroll to the desired tooltip.

Also you can set the Smooth-Scroll plugin parameters one-by-one like offset, speed and easing.

CSS3

Animation are created with CSS3 features so they are really smooth. The tooltips are still gonna work in older browsers, but without animations.

More Features

The plugin has much more features which makes the plugin even more customizeable. There 40 JavaScript parameters which you can set globally, like autoPlay, offset, prompt, etc.!

Calling Mr.Back

The 9 available callbacks (onBeforeLoad, onAfterLoad, onBeforeSwitch, onAfterSwitch, onNext, onPrev, onGo, onFinish, onInit) will help you to create really unique methods or actions, helps you to call custom functions on plugin events, and i didn'T mentioned those plugin methods which you can use to create unique navigations.

In The End

As you can see, it's onto you to take out the maximum from the plugin and we will help you! Please report any kind of bugs, experiences or suggestions in the comments section of the product page, we want to make our script as safe and as bugless as we can!!!
